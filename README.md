ejsonrpc
========

Enhanced JSONRPC WSGI server and client

Features
--------
* JSONRPC v2.0
* HTTP Basic Auth
* Namespaces
* Pythonic function calls. Uses \*args and \*\*kwargs as parameters if you want to, and therefore more flexible for python users.

Planned features
----------------
* Py3k support

Not yet implemented
-------------------
* Batch calls
